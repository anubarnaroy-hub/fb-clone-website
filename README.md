# fb-clone-website

## About the Project

This repository contains a simple Facebook website clone that I built while learning Git and GitHub. The project was created as part of a hands-on Git tutorial to understand how version control works in real-world software development.

The primary goal of this project is not to recreate Facebook with all of its features, but to practice using Git commands and GitHub workflows while working on a frontend project. Throughout the development of this project, I learned how to initialize a Git repository, create commits, work with branches, merge branches, resolve merge conflicts, create tags, and push changes to GitHub.

By using a simple website as the project, I was able to focus on learning Git concepts while also improving my HTML, CSS, and JavaScript skills.
